 Double Dragon
##### Install:
-----------------------------------------------------------------------------------------------------------------------------------------
- clone repository
- run index.html on localhost

##### Simple 2D Beat 'em up game recreated with:
-----------------------------------------------------------------------------------------------------------------------------------------
* Phaser 2.6.2.
* Texture Packer (students version)
* Alfred Spritesheet Unpacker v11
* Double Dragon iOS Sprite Sheets
